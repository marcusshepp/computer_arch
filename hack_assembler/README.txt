An Assembler written in plan Python.

in: HACK assembly

out: HACK machine language

* note *
    - is specific to the HACK machine only.


Requirements:
- Python2.7


Quick Start:

-------------------


Unmodified Hack CPU
`python assem.py <program>.asm`


-------------------


Modified Hack CPU
`python altern_assem.py <program>.asm`
