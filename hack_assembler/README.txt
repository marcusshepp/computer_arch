An Assembler written in plain Python.

in: HACK assembly

out: HACK machine language

* note *
    - is specific to the HACK machine only.


-------------------


Requirements:

- Python2.7


-------------------


Quick Start:

Unmodified Hack CPU
`python Assembler.py <program>.asm`


Modified Hack CPU
`python ModAssembler.py <program>.asm`
